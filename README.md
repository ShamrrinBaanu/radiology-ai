# Radiology AI Triage System

An AI-powered web application for detecting brain hemorrhage from CT scans.

## Features
- Brain hemorrhage detection using ResNet18
- AI attention heatmap visualization
- Emergency triage classification
- AI-generated radiology report
- Patient scan history
- Streamlit dashboard

## Tech Stack
- Python
- PyTorch
- Streamlit
- OpenCV
- SQLite

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
